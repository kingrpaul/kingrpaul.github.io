Esperanto on US-English Keboard With Ubuntu 22.04 HOME

2024_03_10

  1. Install Tweaks from the software store or confirm that it is installed.

  2. Launch Tweaks. From the Keyboard & Mouse section, select Additional Layout Options.

  3. From the Esperanto letters with superscripts selector, select At the corresponding key in a QWERTY layout.

  4. From the Key to choose the 3rd level selector, select only Left Alt and Right Alt never chooses 3rd level.

To type an Esperanto letter that contains a diacritic (ĉ, ĝ, ĥ, ĵ, ŝ, or ŭ), press the Lt-Alt key along with the corresponding letter lacking a diacritic. The Rt-Alt key remains available for alternate functions.
